# Boot-Up
Free Simple OpenSource CMS for blogs or news sites.  

# Installing
1) Import 'db.sql' to your MySQL database  
2) Run install.php to setting-up config file  
3) You are genius!:)  

# Features
1) Bootstrap design  
2) Integrated VK Comments  
3) Category system  
4) HTML tags work in articles
